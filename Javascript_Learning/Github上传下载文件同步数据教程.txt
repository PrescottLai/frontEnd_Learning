——————————————————————————下载————————————————————
git clone  <网址>                      注意下载下来后要进入这个文件夹才能查看分支，切换分支，一般情况下，都默认显示“main”分支

查看分支：git branch

查看所有分支（包括隐藏的）：git branch -a      隐藏的一般是远程的分支 可以用 git branch -r 单独查看

创建分支：git branch <name>

创建本地分支并跟踪远程分支：git checkout -t <主机名字>/<主机名字下的分支名字>  例如：git checkout -t origin/master    该命令会先 track 远程主机的分支 master， 并在本地建立一个于分支同样的名字 master 来 跟踪 远程分支 origin/master

切换分支：git checkout <name>或者git switch <name>

创建+切换分支：git checkout -b <name>或者git switch -c <name>

合并某分支到当前分支：git merge <name>

删除分支：git branch -d <name>

查看所有远程主机：git remote

查看远程主机的网址： git remote -v
